Elevarm BE

## Tech Stack :
- Node Express.js
- MongoDB Atlas
- jwt for auth 

## HOW TO USE
- Git clone this repository
- npm install
- npm run
- REST API is now available in port 3000 (localhost:3000)

## Services
- User service
- Gojek service (User for drivers, and transcation for ride)
- Gofood service (Restaurant and menu service, transaction for nota)
